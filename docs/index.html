BC7215 AC Controller
ESP32-C3 Web Installer
ESP Web Tools Home Assistant ready

Install your BC7215 AC firmware in the browser.

A simple web installer for ESP32-C3 boards using the default BC7215A wiring. Connect the board with USB, install the firmware, configure WiFi, and add the device in Home Assistant.

Step 1

Install firmware

Connect the ESP32-C3 board to your computer with a USB cable, then click the button below.

✓ Use desktop Chrome or Microsoft Edge. Firefox usually does not support this installer.
✓ This page is designed for ESP32-C3 boards with the default GPIO wiring below, including 3.3V VCC.

Hardware

Default wiring

Wire the BC7215A module to the ESP32-C3 before installing the firmware. Connect VCC to 3.3V.

BC7215 module signal ESP32-C3 GPIO
BC7215 RX GPIO3
BC7215 TX GPIO4
BC7215 BUSY GPIO1
BC7215 MOD GPIO0
GND GND
VCC 3.3V

Setup

After installation

After flashing, configure WiFi and add the ESPHome device in Home Assistant.

  1. Configure WiFi when prompted by the installer.
  2. Open Home Assistant.
  3. Go to Settings → Devices & services.
  4. Add the discovered ESPHome device.
  5. Prepare your IR remote to 25°C/Cooling mode.
  6. Open the device page and click Pair Button.
  7. Press Fan Speed on remote to finish pairing.
!
Important: This web installer is only for ESP32-C3 boards using the default wiring. If you use different GPIO pins, build a custom ESPHome YAML instead.
